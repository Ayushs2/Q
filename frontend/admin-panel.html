Excel Upload

Upload multiple MCQs at once using an Excel (.xlsx) file. Make sure your file follows the required format.

Drag & drop your .xlsx file here or click to browse

Required Columns:

Question | Option A | Option B | Option C | Option D | Correct Answer | Explanation | Subject | Topic | Exam | Year | Difficulty | Marks

New Fields Explained:

  • Difficulty: Must be exactly "easy", "medium", or "hard" (default: medium)
  • Marks: Numerical value like 1, 1.5, 2, 2.5, etc. (increments of 0.25, default: 1)

Important Notes:

  • All LaTeX equations must be wrapped in dollar signs: $equation$
  • Correct Answer must be exactly A, B, C, or D
  • Year column is optional but recommended
  • File size should be less than 5MB
  • Maximum 1000 questions per upload
